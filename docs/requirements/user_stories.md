# User Story Template

## Story Format
As a [type of user],
I want [goal/desire]
so that [benefit].

## Example
As a project manager,
I want to see all team member assignments in one view
so that I can quickly assess workload distribution.

## Acceptance Criteria Template
1. Given [context]
   When [action]
   Then [expected outcome]

## Priority Labels
- Must Have
- Should Have
- Could Have
- Won't Have

## Story Points/Effort
- 1 (Trivial)
- 2 (Small)
- 3 (Medium)
- 5 (Large)
- 8 (Very Large)
